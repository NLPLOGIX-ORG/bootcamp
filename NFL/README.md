# NFL Play-by-Play Data 2009-2018
You can find the dataset on Kaggle [here](https://www.kaggle.com/maxhorowitz/nflplaybyplay2009to2016) or in our Dropbox folder.

This is a large file so it cannot be pushed to this repository. Have it downloaded to your local hard drive.

If you are not familiar with the game of football, don't worry! You can a glossary on all of the terms you will be using with this data [here](https://operations.nfl.com/learn-the-game/nfl-basics/terms-glossary/).
